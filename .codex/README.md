# Codex Local Environment Setup (Optional)

Codex stores project-level configuration in a `.codex/` folder. This can include:
- **Setup scripts** that run when Codex creates a new worktree
- **Actions** for common commands (tests, dev server, lint)
- **Config settings** (sandbox, approvals, model defaults)

These settings are created and managed through the Codex app settings pane. If you want to share them with your team, copy the generated files into this `.codex/` folder and commit them to the repo.

Docs:
- https://developers.openai.com/codex/app/local-environments/
- https://developers.openai.com/codex/config-basic/
